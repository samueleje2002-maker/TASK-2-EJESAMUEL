Exploratory Data Analysis Using Microsoft Excel
Project Overview

This project was completed during my DecodeLab Data Analytics Virtual Internship.

The objective was to perform Exploratory Data Analysis (EDA) on a cleaned e-commerce dataset to uncover trends, patterns, and business insights that support data-driven decision-making.

Objectives
Understand the structure of the dataset
Generate descriptive statistics
Analyze revenue performance
Evaluate product performance
Examine order status distribution
Assess referral source performance
Analyze monthly revenue trends
Derive actionable business insights
